# 伪代码判题系统

一个纯前端的本地练习项目，用于按题目编写代码并通过文本差异对比计算“正确率”。项目不依赖后端，直接在浏览器中运行，题目数据与本地进度都保存在当前设备上。

## 功能概览

- 题库首页展示题目列表、提交次数、上次正确率、平均正确率。
- 答题页内置 Monaco Editor，可直接编写和提交代码。
- 提交后使用 `jsdiff` 对用户代码和标准答案做词级差异比较。
- 正确率支持提交后立即修改。
- 提供独立的进度修改页，可查看、修改、删除本机所有提交记录。
- 支持一键将成绩改为 `100%` 或 `0%`。
- 所有正确率统一限制为 `0` 到 `100`，并向下取整为整数。
- 分数按区间使用红、橙、绿渐变显示。
- 支持导出和导入本地进度数据。

## 页面说明

### 1. 首页

文件：`index.html`

- 展示题目列表。
- 可进入答题页。
- 可导出进度、导入进度、进入进度修改页。

### 2. 答题页

文件：`problem.html`

- 左侧显示题目内容。
- 右侧提供代码编辑器。
- 提交后显示差异对比结果和当前正确率。
- 可立即修改本次提交成绩。

### 3. 进度修改页

文件：`progress.html`

- 展示当前设备保存的全部记录。
- 可按单条记录修改分数。
- 可删除单条记录。
- 提供 `✔` 一键改为 `100%`，`✘` 一键改为 `0%`。

## 快速开始

这是一个静态项目，不需要安装依赖。

### 方式一：直接打开

直接用浏览器打开 `index.html` 即可运行。

### 方式二：本地静态服务器

如果你希望用本地服务器访问，也可以在项目目录启动任意静态服务，例如：

```powershell
npx serve .
```

然后在浏览器中打开命令输出的地址。

## 数据与存储

### 题目数据

题目定义位于 `data.js`，当前每道题包含以下字段：

```js
{
  id: "two-sum",
  title: "1. 两数之和 (Two Sum)",
  language: "python",
  descriptionHTML: "...",
  initialCode: "...",
  expectedCode: "..."
}
```

字段说明：

- `id`：题目唯一标识。
- `title`：题目标题。
- `language`：Monaco Editor 使用的语言模式。
- `descriptionHTML`：题面 HTML。
- `initialCode`：编辑器初始代码。
- `expectedCode`：用于对比计算正确率的标准答案。

### 本地进度存储

进度保存在浏览器 `localStorage` 中，键名为：

```text
pseudo_oj_records
```

数据结构示例：

```json
{
  "two-sum": [100, 76, 88],
  "palindrome-number": [0, 52]
}
```

说明：

- 每个数组元素表示该题的一次提交成绩。
- 分数始终为整数。
- 导入的数据会自动归一化到 `0` 到 `100`。

## 正确率规则

- 使用 `Diff.diffWords` 做词级对比。
- 按匹配词数与标准答案词数计算比例。
- 结果统一向下取整。
- 最终分数会被限制在 `0` 到 `100`。

## 扩展方式

### 添加新题目

在 `data.js` 中向 `window.PROBLEM_LIST` 追加一项即可。

### 修改评分逻辑

评分逻辑位于 `problem.html` 中的 `submitCode()`。

### 修改进度规则

分数归一化、颜色映射、记录增删改逻辑位于 `tools.js`。

## 依赖说明

项目当前通过 CDN 使用以下前端库：

- Monaco Editor
- jsdiff

如果后续需要离线使用，可以将这两个依赖改为本地静态资源。
